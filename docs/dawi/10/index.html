O Problema: Páginas "Estáticas"

Até agora, todo o conteúdo das nossas páginas foi escrito diretamente no HTML. Isso é chamado de conteúdo "hard-coded".

Mas e se...

  • ...você tivesse que criar uma página com 100 produtos?
  • ...os dados de um cardápio mudassem toda semana?
  • ...as notícias de um portal precisassem ser atualizadas a cada hora?

Copiar e colar o HTML para cada item seria inviável. Precisamos de uma forma de separar os dados da apresentação.

A Solução: JSON

JSON significa JavaScript Object Notation. É um formato de texto leve e de fácil leitura para humanos, usado para armazenar e transportar dados.

Pense nele como uma "lista de compras" universal. É um arquivo de texto simples que o JavaScript (e quase qualquer outra linguagem) consegue ler e entender instantaneamente.

// Exemplo de um objeto JSON simples
{
  "nome": "Caneta Mágica",
  "preco": 19.99,
  "em_estoque": true
}

Antes do JSON: Objetos e Arrays em JS

Para entender JSON, primeiro precisamos lembrar de duas estruturas essenciais do próprio JavaScript:

Objetos (ou "Coleções")

Agrupam dados relacionados usando pares chave-valor.

// Um objeto que descreve um livro
const livro = {
  titulo: "O Guia do Mochileiro",
  autor: "Douglas Adams",
  paginas: 208
};

// Acessando um valor:
console.log(livro.titulo);
// Saída: "O Guia do Mochileiro"
Arrays (ou "Listas")

Armazenam uma lista ordenada de itens.

// Um array com uma lista de tarefas
const tarefas = [
  "Estudar JavaScript",
  "Fazer o exercício",
  "Tomar um café"
];

// Acessando um item (pela posição):
console.log(tarefas[0]);
// Saída: "Estudar JavaScript"

Conexão: JSON é basicamente a representação em texto dessas mesmas estruturas!

A Estrutura do JSON

O JSON é construído com duas estruturas básicas:

  • Objetos { }: Uma coleção de pares "chave":"valor". A chave é sempre um texto entre aspas duplas.
  • Arrays [ ]: Uma lista ordenada de valores.
// Um array de objetos
[
  {
    "id": 1,
    "produto": "Poção de Cura",
    "ingredientes": ["Água Benta", "Erva Rara", "Pó de Fada"]
  },
  {
    "id": 2,
    "produto": "Amuleto da Sorte",
    "ingredientes": ["Trevo de 4 Folhas", "Pedra da Lua"]
  }
]

Como o JavaScript lê o arquivo?

A API fetch()

Usamos a função fetch() do JavaScript para fazer requisições de rede. Neste caso, vamos "requisitar" nosso arquivo local criaturas.json.

O fetch() é assíncrono. Isso significa que ele não trava a página enquanto espera o arquivo carregar. Ele faz o pedido e "promete" que vai nos dar uma resposta quando terminar.

Analogia: É como pedir uma pizza. Você liga (fetch), faz o pedido, e continua fazendo suas coisas. Você não fica parado olhando para a porta. Quando a pizza chega (a "promessa" é resolvida), você a pega e come.

Trabalhando com a "Promessa" (Promise)

Como o fetch() nos retorna uma promessa, usamos os métodos .then() e .catch() para lidar com o resultado.

.then(resposta => { ... })

"ENTÃO, quando o arquivo for encontrado, pegue a resposta..."

.then(dados => { ... })

"...e ENTÃO, quando os dados da resposta forem processados, use os dados."

.catch(erro => { ... })

"CASO dê algum erro no processo (ex: arquivo não encontrado), me avise."

Carregando e Vendo os Dados

Este é o código completo para carregar nosso arquivo JSON e exibir os dados no console do navegador.

fetch('criaturas.json')
  .then(response => response.json()) // 1. Converte a resposta em JSON
  .then(data => {                   // 2. Usa os dados convertidos
    console.log(data); // Exibe o array de criaturas no console
  })
  .catch(error => {
    console.error('Houve um erro ao carregar o arquivo:', error);
  });

Crie um arquivo criaturas.json no mesmo diretório do seu HTML. Cole o código JSON do slide 6 nele. Depois, adicione o código JavaScript acima ao seu arquivo .js. Abra o console do navegador e veja a mágica!

Percorrendo Dados com .forEach()

Uma vez que temos um array de dados (vindo do JSON), a forma mais moderna e legível de passar por cada item é usando o método .forEach().

Ele executa uma função para cada elemento do array, um por um.

const criaturas = [
  { nome: "Grifo" },
  { nome: "Dragão" },
  { nome: "Elfo" }
];

// Para cada 'criatura' dentro do array 'criaturas'...
criaturas.forEach(criatura => {
  // ...execute este código.
  console.log(`Criatura encontrada: ${criatura.nome}`);
});

// Saída no console:
// Criatura encontrada: Grifo
// Criatura encontrada: Dragão
// Criatura encontrada: Elfo

Usaremos o .forEach() para criar um card HTML para cada criatura do nosso arquivo JSON.

Ok, temos os dados. E agora?

Agora vem a parte mais legal: vamos usar o JavaScript para, dinamicamente, criar o HTML para cada criatura e injetá-lo na nossa página.

O plano:

  1. Carregar os dados com fetch().
  2. Percorrer o array de criaturas com forEach.
  3. Para cada criatura, criar uma string de texto contendo o HTML de um card do Bootstrap.
  4. Juntar todas essas strings.
  5. Colocar a string gigante de HTML dentro de um <div> na nossa página.

Juntando Tudo: O Exemplo Completo

Veja como o código HTML e JavaScript interagem para criar a página dinamicamente.

HTML (index.html)
<!-- Container onde os cards serão inseridos -->
<div class="container">
    <div id="grimorio" class="row g-4">
        <!-- Os cards aparecerão aqui! -->
    </div>
</div>
JavaScript (script.js)
fetch('criaturas.json')
  .then(res => res.json())
  .then(criaturas => {
    
    const grimorioDiv = document.getElementById('grimorio');
    let cardsHTML = ''; // String para guardar o HTML

    // Loop para cada criatura no JSON
    criaturas.forEach(criatura => {
      // Usamos template literals (``) para montar o HTML
      cardsHTML += `
        <div class="col-md-4">
          <div class="card h-100">
            <img src="${criatura.imagem}" class="card-img-top">
            <div class="card-body">
              <h5 class="card-title">${criatura.nome}</h5>
              <p class="card-text">${criatura.descricao}</p>
            </div>
             <div class="card-footer">
                <small class="text-muted">Habitat: ${criatura.habitat}</small>
             </div>
          </div>
        </div>
      `;
    });

    // Insere a string de HTML no div
    grimorioDiv.innerHTML = cardsHTML;
  });
