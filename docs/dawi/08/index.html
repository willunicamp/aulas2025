O Coração do Layout: O Sistema de Grid

Se você entender o Grid, você entende 80% do Bootstrap. A lógica é uma hierarquia de 3 níveis:

  1. .container: A caixa principal que centraliza seu conteúdo.
  2. .row: Uma linha horizontal que serve como um contêiner para as colunas.
  3. .col: As colunas verticais dentro de uma linha. Cada linha tem 12 colunas imaginárias.

Responsividade com Breakpoints

O segredo para um site funcionar bem no celular e no desktop é usar os breakpoints do Grid. Eles dizem "em telas de tal tamanho, ocupe X colunas".

<div class="row">
  <!-- Ocupa 12/12 colunas em telas pequenas (celular),
       6/12 em telas médias (tablet) e
       4/12 em telas grandes (desktop) -->
  <div class="col-12 col-md-6 col-lg-4">
    ...
  </div>
</div>
  • col- (Extra small < 576px) - Padrão mobile-first
  • col-sm- (Small ≥ 576px)
  • col-md- (Medium ≥ 768px)
  • col-lg- (Large ≥ 992px)
  • col-xl- (Extra large ≥ 1200px)

Componente: Navbar

Vamos começar a montar nosso portfólio pela navegação principal.

<nav class="navbar navbar-expand-lg bg-body-tertiary">
  <div class="container">
    <a class="navbar-brand" href="#">Meu Portfólio</a>
    <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav">
      <span class="navbar-toggler-icon"></span>
    </button>
    <div class="collapse navbar-collapse" id="navbarNav">
      <ul class="navbar-nav ms-auto">
        <li class="nav-item">
          <a class="nav-link" href="#sobre">Sobre Mim</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#projetos">Projetos</a>
        </li>
      </ul>
    </div>
  </div>
</nav>

Desvendando a Navbar

A Navbar parece complexa, mas é uma combinação de peças que trabalham juntas para a responsividade.

.navbar-toggler
Este é o "botão sanduíche" (hamburger button) que aparece apenas em telas pequenas. Ele é o gatilho para mostrar ou esconder o menu.
.collapse .navbar-collapse
Esta <div> é o contêiner que "esconde" (collapse) os links do menu em telas pequenas. Em telas grandes, ela se expande automaticamente (navbar-expand-lg).
data-bs-toggle="collapse" e data-bs-target="#navbarNav"
Essa é a "mágica" do JavaScript do Bootstrap! O data-bs-toggle diz ao botão que sua função é controlar um collapse. O data-bs-target diz a ele qual elemento colapsar, usando o ID (#navbarNav).

Classes Utilitárias: A "cola" do Design

Os utilitários são classes que fazem uma única coisa, mas fazem muito bem. Eles nos permitem ajustar o design sem escrever CSS.

Espaçamento

m para margem, p para padding. t (top), b (bottom), s (start), e (end), x (horizontal), y (vertical). Os números vão de 0 a 5.

Exemplo: .my-5 (margem no eixo Y), .px-3 (padding no eixo X).

Cores e Background

Classes como .bg-primary, .bg-light, .bg-dark mudam a cor de fundo. .text-white, .text-muted mudam a cor do texto.

Este div tem as classes .bg-dark, .text-white, .p-5 e .text-center

Componente Interativo: Modal

O Modal é uma janela pop-up que aparece sobre o conteúdo da página. É ótimo para exibir mais detalhes sem sair da tela.

O Modal, assim como a Navbar, precisa de duas partes: o gatilho (o botão que o abre) e a estrutura do Modal (o conteúdo que fica escondido).

<!-- 1. O Gatilho (O botão dentro do Card) -->
<button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#exemploModal">
  Ver Detalhes
</button>

<!-- 2. A Estrutura do Modal (pode ficar em qualquer lugar, geralmente no fim do body) -->
<div class="modal fade" id="exemploModal" tabindex="-1">
  <div class="modal-dialog">
    <div class="modal-content">
      <div class="modal-header">
        <h5 class="modal-title">Detalhes do Projeto</h5>
        <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
      </div>
      <div class="modal-body">
        <p>Aqui vão todas as informações detalhadas sobre o projeto.</p>
      </div>
      <div class="modal-footer">
        <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">Fechar</button>
      </div>
    </div>
  </div>
</div>

Formulários: Checks e Radios

Para opções de múltipla escolha ou única escolha, usamos a estrutura .form-check.

<!-- Checkbox -->
<div class="form-check">
  <input class="form-check-input" type="checkbox" value="" id="flexCheckDefault">
  <label class="form-check-label" for="flexCheckDefault">
    Aceito os termos e condições
  </label>
</div>

<!-- Rádio -->
<div class="form-check">
  <input class="form-check-input" type="radio" name="flexRadioDefault" id="flexRadioDefault1">
  <label class="form-check-label" for="flexRadioDefault1">
    Opção 1
  </label>
</div>

Formulários: Select

Para listas de opções (dropdowns), usamos a classe .form-select.

<label for="assuntoSelect" class="form-label">Assunto</label>
<select class="form-select" id="assuntoSelect">

  <option value="1">Orçamento</option>
  <option value="2">Dúvida</option>
  <option value="3">Feedback</option>
</select>

Utilitários: Flexbox

Flexbox é um modelo de layout poderoso. Bootstrap o simplifica com classes utilitárias, perfeitas para alinhar itens em um rodapé.

<div class="d-flex justify-content-between align-items-center">
  <div>Item à Esquerda</div>
  <div>Item à Direita</div>
</div>
  • .d-flex: Ativa o modo flex.
  • .justify-content-*: Alinha horizontalmente (start, end, center, between, around).
  • .align-items-*: Alinha verticalmente (start, end, center).
