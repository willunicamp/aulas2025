Desvendando Layouts Bidimensionais

Flexbox vs. Grid

Flexbox é incrível para alinhar itens em UMA DIMENSÃO (uma linha ou uma coluna).

Grid foi criado para layouts em DUAS DIMENSÕES (linhas E colunas ao mesmo tempo). É o poder supremo para criar a estrutura principal de uma página.

Parte 1: Poderes do Contêiner Grid (Pai)

display: grid

Este é o comando que ativa a magia. Transforma um elemento em contêiner grid e seus filhos diretos em itens grid.

.container { display: grid; }

grid-template-columns/rows

Define o esqueleto do grid. Você pode usar qualquer unidade de CSS, a unidade fr, e funções.

/* Exemplos de Valores */
grid-template-columns: 100px 100px 100px; /* 3 colunas de 100px */
grid-template-columns: 25% 50% 25%;       /* 3 colunas em porcentagem */
grid-template-columns: 1fr 2fr;           /* 2 colunas, a 2ª com o dobro da largura */
grid-template-columns: 100px 1fr auto;    /* Misturando unidades */

A unidade fr (fração) representa uma fração do espaço disponível no contêiner. É a forma mais poderosa de criar layouts flexíveis com Grid.

Exemplo: grid-template-columns

Vamos ver na prática como as colunas são definidas.

.container {
                display: grid;
                gap: 10px;
}

Função repeat()

Uma forma limpa de repetir um padrão de trilhas. Ótimo para grids com muitas colunas ou linhas iguais.

/* Sintaxe: repeat(quantidade, tamanho) */
grid-template-columns: repeat(3, 1fr); /* Equivalente a: 1fr 1fr 1fr */
grid-template-columns: repeat(4, 100px); /* 4 colunas de 100px */
grid-template-columns: repeat(2, 1fr 2fr); /* Padrão: 1fr 2fr 1fr 2fr */

Exemplo: repeat()

Veja como a função repeat() simplifica a criação de grids complexos.

.container {
                display: grid;
                gap: 10px;
}

Função minmax()

Define um tamanho mínimo e máximo para uma trilha. Essencial para responsividade.

/* Sintaxe: minmax(minimo, maximo) */
grid-template-columns: minmax(200px, 1fr); /* Mínimo 200px, máximo 1fr */

/* Combo poderoso para galerias: */
grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));

auto-fit: Cria quantas trilhas couberem no espaço disponível. Se o espaço aumentar, ele cria mais colunas. Se diminuir, ele quebra a linha.

Exemplo: minmax() com auto-fit

Cria um layout de galeria responsivo que ajusta o número de colunas automaticamente.

.container {
                display: grid;
                gap: 10px;
                grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));
}

grid-template-areas

Permite nomear as áreas do grid, criando um "mapa" visual do seu layout. Cada string representa uma linha.

/* CSS do Contêiner (Pai) */
.container {
                display: grid;
                grid-template-areas: 
                "header header"
                "main   sidebar";
}

/* CSS dos Itens (Filhos) */
.header { grid-area: header; }
.main { grid-area: main; }
.sidebar { grid-area: sidebar; }

<!-- HTML -->
<div class="container">
  <div class="header">Header</div>
  <div class="main">Main</div>
  <div class="sidebar">Sidebar</div>
</div>

Entendendo grid-template-areas

A string "main sidebar" define a segunda linha do grid. A palavra main mapeia para a primeira coluna, e sidebar para a segunda. O espaço em branco entre elas é o que separa as colunas.

A grande utilidade é poder reorganizar todo o layout apenas mudando o "mapa" de texto, especialmente em media queries.

grid-template-areas na Prática

Veja como podemos mudar completamente o layout para mobile apenas reescrevendo o "mapa" de áreas.

/* Desktop */
.container {
                grid-template-areas: 
                "header header"
                "main   sidebar";
}

gap

Cria um espaçamento entre as colunas e linhas do grid.

.container {
                display: grid;
                grid-template-columns: 1fr 1fr 1fr;
                gap: 20px;

justify-items e align-items

Alinham os itens dentro de suas células. justify no eixo horizontal, align no eixo vertical.

.container {
                display: grid;
                justify-items: start;
                align-items: start;
}

Parte 2: Poderes dos Itens Grid (Filhos)

grid-column e grid-row

Determina a localização de um item no grid, especificando em quais linhas de grade ele começa e termina. 3 / 5 significa: comece na linha de grade 3 e termine na linha 5 (ocupando as colunas 3 e 4).

.item-A {
                grid-column: 1 / 3;
                grid-row: 1 / 2;
}

A palavra-chave span

Em vez de definir o início e o fim, você pode dizer a um item para "se estender" por um número de trilhas a partir de onde ele estiver.

grid-column: span 3;   /* Ocupa 3 colunas a partir de sua posição inicial */
grid-column: 2 / span 2; /* Começa na linha 2 e ocupa 2 colunas */

Exemplo: span

Veja como span é uma forma flexível de expandir itens.

.item-A {
                grid-column: span 2;
                grid-row: auto;
}

grid-area

Atribui um item a uma área nomeada que você definiu no contêiner com grid-template-areas.

.meu-header { grid-area: header; }
.meu-main { grid-area: main; }
.minha-sidebar { grid-area: sidebar; }
.meu-footer { grid-area: footer; }

Exemplo: grid-area

Clique para atribuir cada item à sua área nomeada no grid.

/* Clique nos botões para aplicar o grid-area */
#area-header
#area-main
#area-sidebar
#area-footer
/* CSS do Contêiner (Pai) */
.container {
                display: grid;
                grid-template-areas: 
                "header header"
                "main   sidebar";
                "footer footer";
}

justify-self e align-self

Permite que um único item ignore o alinhamento definido por justify-items e align-items no contêiner.

.item-rebelde {
                justify-self: start;
                align-self: start;
}
