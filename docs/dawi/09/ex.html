Formulário de Cadastro

🚀 Sua Missão

Seu objetivo é criar um arquivo JavaScript para validar este formulário antes do envio. As regras de validação estão descritas nos passos abaixo. Se houver algum erro, uma mensagem específica deve ser exibida no alerta. Se tudo estiver correto, exiba uma mensagem de sucesso!

📋 Passos para o JavaScript

Em seu arquivo .js, siga estes passos para criar a validação:

  1. Capture o Envio do Formulário:
    • Selecione o elemento do formulário (#cadastroForm).
    • Adicione um "ouvinte de evento" para o evento submit.
    • Dentro da função do evento, use evento.preventDefault() para impedir que a página recarregue.
  2. Crie a Lógica de Validação:
    • Dentro da função de submit, selecione todos os campos do formulário e o div de mensagens.
    • Crie um array vazio chamado erros.
    • Verifique cada regra:
      • Se o nome tiver 10 caracteres ou menos, adicione uma mensagem ao array erros.
      • Se o CPF não tiver exatamente 11 caracteres, adicione outra mensagem.
      • Se nenhum estado for selecionado (o valor for ""), adicione outra mensagem.
      • Se menos de 2 checkboxes de interesse estiverem marcados, adicione outra mensagem.
  3. Exiba o Resultado:
    • Após verificar todas as regras, cheque se o array erros tem algum conteúdo (erros.length > 0).
    • Se houver erros: mostre o div de mensagens, adicione a classe alert-danger e exiba todas as mensagens de erro.
    • Se não houver erros: mostre o div de mensagens, adicione a classe alert-success e exiba uma mensagem de sucesso.
