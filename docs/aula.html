A Missão de Hoje

  • Uma Regra Final para o Chefe: A propriedade `gap`.
  • Os Poderes de Tamanho: `flex-grow`, `flex-shrink`, e `flex-basis`.
  • Os Poderes de Posição: `order` e `align-self`.

Última Regra do Chefe: `gap`

`gap` é a forma moderna e limpa de criar espaçamento entre os itens, direto no contêiner.

.container { display: flex; gap: 20px; }

Atenção: A partir de agora, os poderes são aplicados nos ITENS (filhos)!

Poder Individual #1: `flex-grow`

Define a habilidade de um item crescer e ocupar o espaço livre. O valor é uma proporção de distribuição do espaço que SOBROU.

.item-que-cresce { flex-grow: 1; }

Poder Individual #2: `flex-shrink`

Define a habilidade de um item encolher quando não há espaço. Use `flex-shrink: 0` para protegê-lo de ser "espremido".

Observe: A soma das larguras (200px * 3 = 600px) é maior que o contêiner (500px), então os itens encolhem por padrão.

.item-teimoso { flex-shrink: 0; } /* Este item não vai encolher */

Poder Individual #3: `flex-basis`

Define o tamanho inicial de um item antes do espaço ser distribuído. É o "ponto de partida" para o `flex-grow` e `flex-shrink` agirem.

.item-com-base { flex-basis: 250px; } /* Começa com 250px */
basis: 150px
grow: 1
basis: 200px

O Combo: `flex`

É um atalho para `flex-grow`, `flex-shrink`, e `flex-basis`.

.item { flex: [grow] [shrink] [basis]; }

/* COMUNS: */
flex: 1;      /* Atalho para: flex: 1 1 0%; (cresce e encolhe, base 0) */
flex: auto;   /* Atalho para: flex: 1 1 auto; (cresce e encolhe, baseia-se no conteúdo) */
flex: none;   /* Atalho para: flex: 0 0 auto; (não cresce nem encolhe, rígido) */

Poder Individual #4: `order`

Muda a ordem visual dos itens, sem alterar o HTML. O padrão é `order: 0`. Valores negativos vêm primeiro.

.item-primeiro { order: -1; } .item-ultimo { order: 1; }

Poder Individual #5: `align-self`

Permite que um único item ignore o `align-items` do contêiner e escolha seu próprio alinhamento vertical.

.item-rebelde { align-self: flex-end; }
