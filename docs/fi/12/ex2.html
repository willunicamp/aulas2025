Exercícios Práticos: Condicionais e Laços

Exercício 1: Verificador de Idade

O objetivo é praticar a estrutura if/elif/else e os operadores de comparação numérica.

Sua Missão:

  1. Crie um script chamado verificar_idade.sh.
  2. Peça ao usuário para digitar sua idade e armazene o valor em uma variável.
  3. Use uma estrutura condicional para verificar a idade e exibir uma das seguintes mensagens:
    • Se a idade for menor que 13: "Você é uma criança."
    • Se a idade for entre 13 e 17 (inclusive): "Você é um adolescente."
    • Se a idade for 18 ou maior: "Você é um adulto."
  4. Desafio (opcional): Adicione uma verificação para garantir que o usuário digitou um número e não um texto.

Exercício 2: Contador de Arquivos

Este exercício foca no uso do laço for para iterar sobre itens, combinado com um teste de arquivo (-f) e aritmética básica.

Sua Missão:

  1. Crie um script chamado contar_arquivos.sh.
  2. Inicialize uma variável contadora com o valor 0.
  3. Use um laço for para percorrer todos os itens no diretório atual (dica: use o curinga *).
  4. Dentro do laço, use um if com o teste -f para verificar se o item atual é um arquivo regular (e não um diretório).
  5. Se for um arquivo, incremente a sua variável contadora.
  6. Após o laço terminar, exiba uma mensagem final mostrando o número total de arquivos encontrados. Ex: "O diretório atual contém X arquivos."

Exercício 3: Menu Interativo Simples

O objetivo aqui é criar um programa que continua rodando até que o usuário decida sair, praticando o laço while e a comparação de strings.

Sua Missão:

  1. Crie um script chamado menu.sh.
  2. Crie um laço while que rode "infinitamente" (dica: while true).
  3. Dentro do laço, exiba um menu de opções para o usuário:
    • 1) Listar arquivos
    • 2) Mostrar data e hora atual
    • q) Sair
  4. Use o comando read para capturar a escolha do usuário.
  5. Use uma estrutura if/elif/else para verificar a escolha:
    • Se for '1', execute o comando ls -l.
    • Se for '2', execute o comando date.
    • Se for 'q', exiba "Saindo..." e use o comando break para sair do laço.
    • Caso contrário, mostre uma mensagem de "Opção inválida".
