Exercícios Práticos: Introdução ao Shell Script

Exercício 1: Script de Perfil de Usuário

O objetivo deste exercício é praticar o uso de variáveis e a captura de dados do usuário com o comando read.

Sua Missão:

  1. Crie um script chamado perfil.sh.
  2. O script deve começar com uma mensagem de boas-vindas.
  3. Peça ao usuário para digitar seu nome e armazene em uma variável.
  4. Peça ao usuário para digitar sua idade e armazene em outra variável.
  5. Peça ao usuário para digitar seu hobby favorito e armazene em uma terceira variável.
  6. No final, o script deve exibir uma mensagem de resumo formatada, usando as três variáveis, como no exemplo abaixo:
    "Olá [NOME], que legal que você tem [IDADE] anos e gosta de [HOBBY]!"
  7. Não se esqueça de tornar o script executável com chmod +x perfil.sh.

Exercício 2: Criador Simples de Arquivos

Este exercício foca em combinar variáveis e executar um comando do sistema (touch) com base na entrada do usuário.

Sua Missão:

  1. Crie um script chamado criar_arquivo.sh.
  2. Peça ao usuário para digitar o nome base do arquivo que deseja criar (ex: relatorio).
  3. Peça ao usuário para digitar a extensão do arquivo (ex: txt, md, log).
  4. Crie uma nova variável que junte o nome base, um ponto e a extensão (ex: relatorio.txt).
  5. Use o comando touch com essa nova variável para criar o arquivo vazio.
  6. Exiba uma mensagem de sucesso confirmando a criação do arquivo. Por exemplo:
    "Arquivo 'relatorio.txt' criado com sucesso!"
