Projeto Final - Sistema de Gerenciamento

Professor: William Paiva

Data de Entrega: 29/11/2025 (Apresentação em sala)

1. Objetivo

O objetivo deste projeto é aplicar de forma prática os conceitos de Programação Orientada a Objetos (POO) e arquitetura de software em camadas no desenvolvimento de um sistema de gerenciamento completo.

Os alunos deverão projetar, modelar, construir o banco de dados e codificar uma aplicação Windows Forms (C#) que resolva um problema de negócio, seguindo estritamente a arquitetura UI - DAL - Entidades apresentada em aula.

2. Requisitos Técnicos Obrigatórios

O projeto deve atender aos seguintes requisitos para ser considerado apto para avaliação:

Arquitetura em Camadas

A solução no Visual Studio deve ser criada usando uma "Solução em Branco" e conter três projetos distintos:

  • UI (Aplicativo do Windows Forms)
  • DAL (Biblioteca de Classes para Acesso a Dados)
  • Entidades (Biblioteca de Classes para Modelos de Dados)

As referências entre os projetos devem ser configuradas corretamente (UI depende de DAL e Entidades; DAL depende de Entidades).

Banco de Dados

  • O grupo deve criar seu próprio banco de dados (MySQL, SQL Server, etc.).
  • O banco deve conter no mínimo 5 tabelas.
  • Os relacionamentos entre as tabelas (1:N ou N:M) devem ser implementados usando chaves estrangeiras (FOREIGN KEY).

Aplicação (UI)

  • A aplicação deve ser desenvolvida em Windows Forms (.NET Framework).
  • Deve haver uma tela principal (Menu, MDI Form, ou Dashboard) que sirva como ponto de entrada para as outras funcionalidades.
  • As demais janelas (formulários de cadastro, consulta, etc.) devem ser abertas a partir desta tela principal.
  • A aplicação deve implementar as operações básicas de CRUD (Cadastrar, Ler, Atualizar, Excluir) para as principais entidades do sistema.

3. Formação dos Grupos

O auxílio de ferramentas de IA (GitHub Copilot, Gemini, ChatGPT, etc.) está liberado e encorajado para aumentar a produtividade.

No entanto, a avaliação será baseada na compreensão do código gerado, e não apenas no seu funcionamento. "Não sei, foi a IA que fez" não será uma resposta aceita e implicará em nota zero na pergunta.
