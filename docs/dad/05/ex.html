Gerenciador de Músicas

Objetivo: Desenvolver uma aplicação para gerenciar uma coleção de músicas.

Instruções Gerais

  • Crie uma nova solução no Visual Studio.
  • Utilize C# com Windows Forms e MySQL.
  • Controles permitidos: TextBox, Label, Button, ComboBox, NumericUpDown, ListBox e MessageBox.

Passo 1: Banco de Dados

Crie a tabela abaixo em um banco de dados existente:

-- Cria a tabela de musicas
CREATE TABLE musicas (
    id INT PRIMARY KEY AUTO_INCREMENT,
    titulo VARCHAR(100) NOT NULL,
    duracao_segundos INT,
    ano_lancamento INT,
    artista VARCHAR(100)
);

Passo 2: Interface

Crie um formulário com: Campos para Título, Duração (s) e Ano, um ComboBox para Artista, um Button "Salvar" e um ListBox para exibir as músicas.

Passo 3: ComboBox

Popule o ComboBox de Artistas manualmente com 3 opções (Ex: "Queen", "Legião Urbana", "Daft Punk").

Passo 4: Create

Implemente o botão "Salvar". Ele deve inserir uma nova música na tabela, usando o texto do artista selecionado. Mostre uma MessageBox de sucesso.

Passo 5: Read

Popule o ListBox com os títulos das músicas. Crie uma classe MusicaListBoxItem com id e titulo, e sobrescreva `ToString()`. A lista deve carregar ao abrir o form e **recarregar** após salvar uma nova música.
