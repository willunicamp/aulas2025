Exercícios Práticos: Ponteiros em C

Pratique a declaração e o uso de ponteiros para manipular a memória.

1. Informações da Memória

Crie um programa que declare uma variável inteira, atribua um valor a ela e declare um ponteiro para inteiro. Faça o ponteiro apontar para a variável e, em seguida, exiba na tela:

  • O valor da variável.
  • O endereço da variável.
  • O valor que o ponteiro está guardando (que deve ser o mesmo endereço da variável).
  • O valor apontado pelo ponteiro.

2. Modificação Indireta

Crie um programa que declare uma variável idade e atribua um valor a ela. Crie um ponteiro pIdade que aponte para idade. Exiba a idade. Em seguida, usando apenas o ponteiro, altere o valor da idade para um novo valor. Exiba a idade novamente para confirmar que a alteração funcionou.

3. Soma através de Ponteiros

Declare duas variáveis inteiras, num1 e num2. Peça ao usuário para digitar valores para elas. Crie dois ponteiros, ptr1 e ptr2, apontando para num1 e num2, respectivamente. Crie uma terceira variável, soma, e faça com que ela receba a soma dos valores apontados pelos ponteiros. Exiba o resultado.

4. Ponteiros e Condicionais

Declare uma variável saldo do tipo float e inicie-a com o valor 1000.0. Crie um ponteiro para ela. Peça ao usuário para digitar um valor de saque. Usando um if, verifique se o saldo (acessado através do ponteiro) é suficiente para o saque. Se for, subtraia o valor do saque do saldo (usando o ponteiro). No final, exiba o saldo final.

5. Ponteiro para Struct (Acesso com (*).)

Crie uma struct Produto com os campos int codigo e float preco. Na main, crie uma variável produto1 e atribua valores a ela. Crie um ponteiro ptrProduto que aponte para produto1. Exiba o código e o preço do produto acessando os membros através do ponteiro, usando a sintaxe com parênteses e ponto: (*ptrProduto).codigo.

6. Ponteiro para Struct (Acesso com Seta ->)

Usando a mesma struct Produto do exercício anterior, crie uma variável produto2 e um ponteiro para ela. Atribua valores aos membros da struct usando apenas o ponteiro e o operador seta (->). Por fim, exiba os dados para confirmar que foram salvos corretamente.

7. Ordem na Memória

Declare duas variáveis, var1 e var2. Crie dois ponteiros, ptr1 e ptr2, que apontem para elas, respectivamente. Compare os **endereços** de memória armazenados nos ponteiros e exiba uma mensagem indicando qual variável está armazenada em um endereço de memória "menor" (vem antes na memória).

8. Atualizando Ficha de Personagem

Crie uma struct Personagem com os campos char nome[50], int vida e int mana. Na main, crie um personagem e preencha seus dados iniciais. Crie um ponteiro para este personagem. Em seguida, simule um "level up" usando **apenas o ponteiro**: aumente a vida em 50 e a mana em 25. Exiba os dados do personagem antes e depois do "level up" para mostrar as alterações.
