Exercícios Práticos: Structs em C

Pratique a criação e manipulação de estruturas para organizar seus dados.

1. Ficha de Livro

Crie um programa que cadastre os dados de um único livro (título, autor e ano de publicação) usando uma struct e, em seguida, exiba os dados cadastrados.

Passos Sugeridos:

  1. Defina uma struct Livro com os campos: char titulo[100], char autor[100] e int ano.
  2. Dentro da função main, declare uma variável do tipo struct Livro, por exemplo, meuLivro.
  3. Peça ao usuário para digitar o título, o autor e o ano do livro, guardando cada informação no membro correspondente (ex: meuLivro.ano).
  4. Após o cadastro, mostre na tela os dados que foram armazenados na sua variável struct.

Exemplo de Funcionamento:

Digite o titulo do livro: O Senhor dos Aneis
Digite o autor do livro: J.R.R. Tolkien
Digite o ano de publicacao: 1954

--- Ficha do Livro ---
Titulo: O Senhor dos Aneis
Autor: J.R.R. Tolkien
Ano de Publicacao: 1954

2. Cadastro de Contato

Crie uma struct para representar um contato de uma agenda, com nome e telefone. Peça ao usuário para preencher os dados de um contato e exiba-os.

Passos Sugeridos:

  1. Defina uma struct Contato com os campos char nome[50] e char telefone[20].
  2. Na main, crie uma variável novoContato do tipo struct Contato.
  3. Solicite ao usuário o nome e o telefone, armazenando nas variáveis corretas (novoContato.nome e novoContato.telefone).
  4. Mostre os dados formatados na tela.

Exemplo de Funcionamento:

Digite o nome do contato: Ana Silva
Digite o telefone do contato: (11) 98765-4321

--- Contato Salvo ---
Nome: Ana Silva
Telefone: (11) 98765-4321

3. Ficha de Vários Filmes

Crie um programa que guarde as informações de três filmes, um de cada vez. Cada filme tem: título, diretor e uma nota de avaliação (de 0 a 10). O programa deve ler os dados dos três filmes e depois exibi-los.

Passos Sugeridos:

  1. Defina uma struct Filme com os campos: char titulo[100], char diretor[100] e float nota.
  2. Na main, declare três variáveis separadas: filme1, filme2 e filme3.
  3. Peça ao usuário para digitar os dados do primeiro filme e guarde-os em filme1.
  4. Repita o processo para o segundo filme (guardando em filme2) e para o terceiro (em filme3).
  5. Ao final, exiba os dados dos três filmes de forma organizada.

Exemplo de Funcionamento:

-- CADASTRO DO 1o FILME --
Digite o titulo: Jurassic Park
Digite o diretor: Steven Spielberg
Digite a nota: 9.3

-- CADASTRO DO 2o FILME --
Digite o titulo: Matrix
Digite o diretor: Wachowskis
Digite a nota: 9.0

-- CADASTRO DO 3o FILME --
Digite o titulo: Interestelar
Digite o diretor: Christopher Nolan
Digite a nota: 9.5

--- FILMES CADASTRADOS ---
1. Jurassic Park, por Steven Spielberg - Nota: 9.3
2. Matrix, por Wachowskis - Nota: 9.0
3. Interestelar, por Christopher Nolan - Nota: 9.5

4. Comparando Personagens

Crie uma struct para um personagem de jogo com os campos: nome, vida e ataque. Crie duas variáveis do tipo personagem. Preencha os dados dos dois personagens e, no final, diga qual deles tem o maior poder de ataque.

Passos Sugeridos:

  1. Defina a struct Personagem (char nome[50], int vida, int ataque).
  2. Na main, declare duas variáveis: personagem1 e personagem2.
  3. Peça ao usuário para preencher todos os dados do personagem1.
  4. Peça ao usuário para preencher todos os dados do personagem2.
  5. Use uma estrutura if/else para comparar personagem1.ataque com personagem2.ataque.
  6. Exiba o nome do personagem que possui o maior ataque.

Exemplo de Funcionamento:

-- Cadastro do Personagem 1 --
Digite o nome: Mago
Digite a vida: 80
Digite o ataque: 150

-- Cadastro do Personagem 2 --
Digite o nome: Guerreiro
Digite a vida: 120
Digite o ataque: 95

O personagem com maior ataque e: Mago

5. Desafio: Estoque da Loja

Este é um desafio para prepará-lo para a próxima aula!

Até agora, criamos apenas uma "ficha" por vez. Mas e se precisarmos cadastrar vários itens? Para isso, podemos criar um vetor de structs!

Crie um programa que cadastre 3 produtos em um estoque. Cada produto tem código, nome e preço. Ao final, o programa deve listar todos os produtos cadastrados.

Passos Sugeridos:

  1. Defina uma struct Produto com os campos: int codigo, char nome[50] e float preco.
  2. Na main, declare um vetor de structs: struct Produto estoque[3];.
  3. Crie um laço for que se repita 3 vezes (de i=0 até i=2).
  4. Dentro do laço, peça ao usuário para digitar o código, nome e preço do produto, guardando nas posições corretas do vetor. Exemplo: scanf("%d", &estoque[i].codigo);.
  5. Após o primeiro laço, crie um segundo laço for para percorrer o vetor e exibir os dados de todos os 3 produtos.

Exemplo de Funcionamento:

-- Cadastro do Produto 1 --
Digite o codigo: 101
Digite o nome: Teclado
Digite o preco: 80.50

-- Cadastro do Produto 2 --
Digite o codigo: 102
Digite o nome: Mouse
Digite o preco: 40.00

-- Cadastro do Produto 3 --
Digite o codigo: 205
Digite o nome: Monitor
Digite o preco: 550.00

--- Lista de Produtos em Estoque ---
Produto 1: Codigo 101, Nome: Teclado, Preco: R$ 80.50
Produto 2: Codigo 102, Nome: Mouse, Preco: R$ 40.00
Produto 3: Codigo 205, Nome: Monitor, Preco: R$ 550.00
