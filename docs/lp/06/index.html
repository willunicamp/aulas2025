O Problema: Dados Espalhados

Imagine que precisamos guardar os dados de um aluno. Usando o que já sabemos, faríamos algo assim:

// Dados do Aluno 1
char nome_aluno1[50] = "Ana";
int ra_aluno1 = 12345;
float nota_aluno1 = 8.5;
// Dados do Aluno 2...
char nome_aluno2[50] = "Beto";
int ra_aluno2 = 54321;
float nota_aluno2 = 7.0;

Funciona, mas é confuso e difícil de gerenciar. As informações que pertencem à mesma "coisa" (o aluno) estão em variáveis separadas.

A Solução: Agrupar com struct

Uma struct (estrutura) nos permite criar nosso próprio tipo de dado personalizado. Pense nela como uma "ficha cadastral" ou um "molde".

Molde: struct Aluno

Nome: _______________

RA: _______________

Nota: _______________

O Molde (A Definição da Struct)

→

aluno1

Nome: "Ana"

RA: 12345

Nota: 8.5

aluno2

Nome: "Beto"

RA: 54321

Nota: 7.0

aluno3

Nome: "Carla"

RA: 98765

Nota: 9.2

Os Dados (As Variáveis do tipo Struct)

A Sintaxe: A Receita do Bolo

Definir uma struct é como criar uma receita. Você diz quais "ingredientes" (variáveis) ela terá. A palavra-chave é struct.

// Definindo o "molde" para um Aluno
struct Aluno {
    char nome[50];
    int ra;
    float nota;
}; // O ponto e vírgula no final é obrigatório!

Regra de Ouro: A definição do molde da struct deve ser feita fora da função main, para que todo o programa saiba que esse novo tipo existe.

Do Molde à Realidade: Criando uma Variável

Depois de definir o molde, podemos declarar variáveis desse novo tipo. A sintaxe é struct NomeDoMolde nome_da_variavel;

// Primeiro, definimos o molde (fora da main)
struct Aluno {
    char nome[50];
    int ra;
    float nota;
};
int main() {
    // Agora, criamos uma variável do tipo Aluno
    // Pense nisso como pegar uma ficha cadastral em branco
    struct Aluno aluno1;
    return 0;
}

Pausa: Como Copiar Textos (Strings)?

Em C, não podemos copiar um texto para um vetor de char usando o sinal de igual =. Isso não funciona:

char nome[50];
nome = "Ana Silva"; // ERRADO! Isso não funciona em C.

Para isso, usamos a função strcpy (string copy), que vive na biblioteca <string.h>. Ela copia o texto da origem para o destino.

#include <string.h> // Precisamos incluir esta biblioteca
char nome[50];
strcpy(nome, "Ana Silva"); // CORRETO!

Voltando a Structs...

Acesso aos Membros: O Operador Ponto (.)

strcpy(aluno1.nome, "Ana Silva");


aluno1.ra = 12345;


aluno1.nota = 8.5;

Usamos o ponto (.) para acessar um campo (membro) específico da nossa variável. A sintaxe é sempre variavel.membro.

struct Aluno aluno1;
nome: "Ana Silva"
ra: 12345
nota: 8.5

Exemplo 1: Cadastro de Aluno

#include <stdio.h>
#include <string.h>
// 1. Definição do molde (fora da main)
struct Aluno {
    char nome[50];
    int ra;
    float nota;
};
int main() {
    // 2. Criação da variável
    struct Aluno aluno1;
    // 3. Atribuição de valores aos membros
    printf("Digite o nome do aluno: ");
    scanf("%49[^\\n]", aluno1.nome); // Não usamos & em 'nome' pois
                               // um vetor já é um endereço!
    printf("Digite o RA: ");
    scanf("%d", &aluno1.ra);
    printf("Digite a nota: ");
    scanf("%f", &aluno1.nota);
    // 4. Exibição dos dados
    printf("\\n--- Ficha do Aluno ---\\n");
    printf("Nome: %s\\n", aluno1.nome);
    printf("RA: %d\\n", aluno1.ra);
    printf("Nota: %.1f\\n", aluno1.nota);
    return 0;
}

💡 Pratique! Copie este código, compile e execute para ver o resultado.

Exemplo 2: Produto de uma Loja

#include <stdio.h>
#include <string.h>
struct Produto {
    int codigo;
    char nome[50];
    double preco;
};
int main() {
    struct Produto prod1;
    prod1.codigo = 101;
    strcpy(prod1.nome, "Caneta Azul");
    prod1.preco = 1.99;
    printf("--- Detalhes do Produto ---\\n");
    printf("Codigo: %d\\n", prod1.codigo);
    printf("Nome: %s\\n", prod1.nome);
    printf("Preco: R$ %.2f\\n", prod1.preco);
    return 0;
}

💡 Pratique! Copie este código, compile e execute para ver o resultado.

Exemplo 3: Personagem de Jogo

#include <stdio.h>
#include <string.h>
struct Personagem {
    char nome[50];
    int vida;
    int forca;
};
int main() {
    struct Personagem heroi;
    strcpy(heroi.nome, "Aragorn");
    heroi.vida = 100;
    heroi.forca = 15;
    printf("%s encontrou um orc!\\n", heroi.nome);
    
    heroi.vida = heroi.vida - 25; // Levou dano
    printf("Vida restante de %s: %d\\n", heroi.nome, heroi.vida);
    return 0;
}

💡 Pratique! Copie este código, compile e execute para ver o resultado.

Copiando Dados: Atribuição de Structs

É possível copiar todos os membros de uma variável struct para outra (do mesmo tipo) com um simples sinal de igualdade (=).

aluno1
nome: "Ana"
ra: 12345
nota: 8.5
→
aluno2
nome: "Ana"
ra: 12345
nota: 8.5
aluno2 = aluno1;

Exemplo 4: Copiando Fichas

#include <stdio.h>
#include <string.h>
struct Aluno {
    char nome[50];
    int ra;
};
int main() {
    struct Aluno aluno1;
    strcpy(aluno1.nome, "Carlos");
    aluno1.ra = 77889;
    // Cria um segundo aluno e copia os dados do primeiro
    struct Aluno aluno2 = aluno1;
    printf("--- Dados do Aluno 2 (copiados) ---\\n");
    printf("Nome: %s\\n", aluno2.nome);
    printf("RA: %d\\n", aluno2.ra);
    return 0;
}

💡 Pratique! Copie este código, compile e execute para ver o resultado.

Resumo da Aula

  • struct é a palavra-chave para criar um novo tipo de dado que agrupa variáveis relacionadas.
  • Primeiro, nós definimos o molde da struct, geralmente fora da main.
  • Depois, nós declaramos variáveis daquele tipo (Ex: struct Aluno aluno1;).
  • Usamos o operador ponto (.) para acessar e modificar os membros de uma variável struct (Ex: aluno1.nota = 10;).
  • Podemos copiar todos os dados de uma struct para outra com uma simples atribuição (aluno2 = aluno1;).
