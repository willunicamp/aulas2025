Bônus: Resolvendo a Leitura de Strings

O scanf("%s", ...); para no primeiro espaço. Vamos ver 3 formas de resolver isso e os problemas de "buffer" (lixo de entrada).

1. Limpando o "buffer" com getchar()

Após um scanf numérico, o "Enter" (\n) fica na entrada. O getchar() consome esse caractere antes de lermos a string.

scanf("%d", &idade);
getchar(); // Limpa o \n
scanf("%[^\n]", nome);

2. O truque do espaço no scanf

Colocar um espaço antes do % no formato do scanf instrui a função a ignorar qualquer "espaço em branco" (incluindo o \n) que ficou para trás.

scanf("%d", &idade);
scanf(" %[^\n]", nome);
//   ^--- Espaço mágico!

3. A forma mais recomendada: fgets

fgets é a função mais segura, pois limita o número de caracteres, evitando que o programa "quebre" se o usuário digitar um texto muito longo.

fgets(nome, 50, stdin);
// Lê até 49 chars
// do teclado (stdin)
// para a variável 'nome'.

Detalhes do fgets: Lendo Tudo com Segurança

fgets sempre lê uma linha inteira como texto. Para ler números, precisamos de um passo extra: ler como texto e depois converter.

Para isso, usamos as funções da biblioteca <stdlib.h>:

  • atoi(): Converte uma string para um inteiro (ASCII to integer).
  • atof(): Converte uma string para um float/double (ASCII to float).

Parâmetros de fgets(buffer, 50, stdin):

  • buffer: A variável char[] que vai receber o texto.
  • 50: O tamanho máximo do buffer (evita invasão de memória).
  • stdin: Significa standard input (entrada padrão), ou seja, o teclado.
#include <stdlib.h> // Biblioteca para atoi e atof
#include <stdio.h>
int main() {
    char buffer[50];
    int idade;
    float altura;
    printf("Digite sua idade: ");
    fgets(buffer, 50, stdin);
    idade = atoi(buffer); // Converte o texto lido para int
    printf("Digite sua altura: ");
    fgets(buffer, 50, stdin);
    altura = atof(buffer); // Converte o texto lido para float
    return 0;
}

Recapitulando: O Incômodo de Muitas Variáveis

Na última aula, vimos que usar uma struct é ótimo para organizar os dados de um item. Mas para cadastrar 3 alunos, o código ainda fica repetitivo e inflexível:

struct Aluno { ... };
int main() {
    struct Aluno aluno1;
    struct Aluno aluno2;
    struct Aluno aluno3;
    // Preencher dados de aluno1...
    // Preencher dados de aluno2...
    // Preencher dados de aluno3...
}

E se precisarmos de 30 alunos? Ou 100? Declarar uma variável para cada um é impraticável.

Uma Solução Antiga: Vetores Paralelos

Uma forma de resolver isso sem structs seria criar um vetor para cada informação. Os dados do aluno na posição [0] estariam espalhados pelos dois vetores.

char nomes[3][50];
int ras[3];
float notas[3];
// O aluno 1 é nomes[0], ras[0], notas[0]
// O aluno 2 é nomes[1], ras[1], notas[1]
// ...

Problema: Os dados estão logicamente conectados, mas fisicamente separados. É fácil cometer erros e misturar as informações. Não é uma boa prática!

A Solução Perfeita: Vetor de Structs

A solução ideal combina o melhor dos dois mundos: o poder de agrupamento das structs com a capacidade de armazenamento dos vetores.

Criamos um único vetor onde cada posição guarda uma "ficha de aluno" completa!

// Declaramos um vetor chamado "turma" que pode guardar 3 Alunos
struct Aluno turma[3];

Sintaxe de Acesso

Para acessar um membro, primeiro usamos os colchetes [] para escolher a "ficha" (o aluno), e depois o ponto . para escolher o "campo" (a informação).

                        
strcpy(turma[0].nome, "Ana");
turma[0].ra = 54321;
turma[0].nota = 9.2;
strcpy(turma[1].nome, "Maria");
turma[1].ra = 22222;
turma[1].nota = 9.8;
strcpy(turma[2].nome, "Jéssica");
turma[2].ra = 12345;
turma[2].nota = 7.7;

A sintaxe é sempre vetor[indice].membro.

i=0
nome: ...
ra: ...
nota: ...
i=1
nome: ...
ra: ...
nota: ...
i=2
nome: ...
ra: ...
nota: ...

Exemplo 1: Cadastrando a Turma

                    
#include <stdio.h>
#include <string.h>
#define TAM 3
struct Aluno {
    char nome[50];
    int ra;
};
int main() {
    struct Aluno turma[TAM];
    // Laço para CADASTRAR os 3 alunos
    for (int i = 0; i < TAM; i++) {
        printf("\n-- Aluno %d --\n", i + 1);
        printf("Nome: ");
        scanf(" %49[^\n]", turma[i].nome);
        printf("RA: ");
        scanf("%d", &turma[i].ra);
    }
    // Laço para EXIBIR os 3 alunos
    printf("\n--- Alunos Cadastrados ---\n");
    for (int i = 0; i < TAM; i++) {
        printf("Aluno: %s | RA: %d\n", turma[i].nome, turma[i].ra);
    }
    return 0;
}

💡 Pratique! Este é o exemplo mais importante da aula. Compile e execute!

Exemplo 2: Produto mais caro do Estoque

Este programa cadastra 3 produtos e, depois de cadastrar, percorre o vetor para descobrir qual deles tem o maior preço.

#include <stdio.h>
#include <string.h>
#define TAM 3
struct Produto {
    char nome[50];
    double preco;
};
int main() {
    struct Produto estoque[TAM];
    int indiceMaisCaro = 0;
    // 1. Laço para preencher o estoque
    for(int i = 0; i < TAM; i++) {
        printf("\n-- Produto %d --\n", i + 1);
        printf("Nome: ");
        scanf(" %49[^\n]", estoque[i].nome);
        printf("Preco: ");
        scanf("%lf", &estoque[i].preco);
    }
    // 2. Laço para encontrar o mais caro
    for (int i = 1; i < TAM; i++) {
        if (estoque[i].preco > estoque[indiceMaisCaro].preco) {
            indiceMaisCaro = i;
        }
    }
    // 3. Exibe o resultado
    printf("\nProduto mais caro: %s\n", estoque[indiceMaisCaro].nome);
    printf("Preco: R$ %.2f\n", estoque[indiceMaisCaro].preco);
    return 0;
}

Exemplo 3: Filtrando Personagens do Jogo

Este programa cadastra uma equipe de 3 personagens e depois exibe apenas os que têm mais de 100 pontos de vida.

#include <stdio.h>
#include <string.h>
#define TAM 3
struct Personagem {
    char nome[50];
    int vida;
};
int main() {
    struct Personagem equipe[TAM];
    // 1. Laço para cadastrar a equipe
    for(int i = 0; i < TAM; i++) {
         printf("\n-- Personagem %d --\n", i + 1);
        printf("Nome: ");
        scanf(" %49[^\n]", equipe[i].nome);
        printf("Vida: ");
        scanf("%d", &equipe[i].vida);
    }
    // 2. Laço para filtrar e exibir
    printf("\n--- Personagens com mais de 100 de Vida ---\n");
    for (int i = 0; i < TAM; i++) {
        if (equipe[i].vida > 100) {
            printf("- %s (Vida: %d)\n", equipe[i].nome, equipe[i].vida);
        }
    }
    return 0;
}

Resumo da Aula

  • Usar múltiplas variáveis de struct ou vetores paralelos não é uma boa prática.
  • A solução ideal para guardar uma coleção de "fichas" é um vetor de structs.
  • Declaramos com struct NomeDoMolde nome_do_vetor[tamanho];
  • Para acessar um membro, usamos o índice do vetor primeiro e depois o operador ponto: vetor[i].membro.
  • Laços for são essenciais para percorrer um vetor de structs, seja para cadastrar, exibir ou processar os dados.
