Exercícios Práticos: Modularização em C

Pratique a separação de código em arquivos .h e .c para criar projetos organizados.

1. Módulo de Conversão de Temperatura

Crie um módulo chamado temperatura que forneça funções para converter Celsius para Fahrenheit e vice-versa. O programa principal deve usar este módulo para realizar as conversões.

Instruções:

  1. Arquivo temperatura.h: Crie este arquivo com os protótipos das funções float celsius_para_fahrenheit(float celsius); e float fahrenheit_para_celsius(float fahrenheit);. Use "include guards".
    (Fórmulas: F = C * 1.8 + 32; C = (F - 32) / 1.8)
  2. Arquivo temperatura.c: Implemente as duas funções de conversão. Lembre-se de incluir "temperatura.h".
  3. Arquivo main.c: Crie o programa principal que inclui <stdio.h> e "temperatura.h". Nele, teste as duas funções com valores fixos (ex: 30°C e 86°F) e imprima os resultados para verificar se estão corretos.
  4. Compilação: Compile seu projeto com o comando gcc main.c temperatura.c -o programa.

2. Módulo de Gerenciamento de Aluno

Crie um módulo aluno para gerenciar os dados de um aluno. O módulo deve ter uma função para preencher os dados de um aluno e outra para exibi-los.

Instruções:

  1. Arquivo aluno.h: Defina uma struct Aluno com os campos char nome[50] e float nota. Declare os protótipos das funções: void preencher_aluno(struct Aluno *a); e void exibir_aluno(struct Aluno a);.
  2. Arquivo aluno.c: Implemente as funções declaradas. A função preencher_aluno deve receber um ponteiro para a struct e usar o operador seta (->) para preencher os dados.
  3. Arquivo main.c: No programa principal, declare uma variável do tipo struct Aluno. Chame a função preencher_aluno, passando o endereço da sua variável. Depois, chame a função exibir_aluno para mostrar os dados.
  4. Compilação: Compile com gcc main.c aluno.c -o programa.

3. Módulo de Controle de Conta Bancária

Crie um módulo conta que gerencie uma conta bancária simples. A conta deve ter um número e um saldo. As funções devem permitir depositar e sacar dinheiro.

Instruções:

  1. Arquivo conta.h: Defina uma struct Conta com int numero e double saldo. Declare os protótipos das funções void depositar(struct Conta *c, double valor); e void sacar(struct Conta *c, double valor);.
  2. Arquivo conta.c: Implemente as funções. A função sacar deve verificar se há saldo suficiente antes de realizar a operação.
  3. Arquivo main.c: Crie uma variável struct Conta e inicialize seu saldo (ex: 1000.0). Exiba o saldo inicial. Chame a função depositar com um valor. Chame a função sacar com outro valor. Exiba o saldo final.
  4. Compilação: Compile com gcc main.c conta.c -o programa.

4. Módulo de Vetor

Crie um módulo vetor_utils (utilidades de vetor) que ofereça funções para manipular um vetor de inteiros.

Instruções:

  1. Arquivo vetor_utils.h: Declare os protótipos de três funções: void preencher_vetor(int vetor[], int tamanho);, void exibir_vetor(int vetor[], int tamanho); e int somar_elementos(int vetor[], int tamanho);.
  2. Arquivo vetor_utils.c: Implemente as três funções. A primeira deve usar um laço para pedir os valores ao usuário. A segunda deve usar um laço para exibir os valores. A terceira deve calcular e retornar a soma dos elementos.
  3. Arquivo main.c: Crie um vetor de inteiros (ex: tamanho 5). Chame a função preencher_vetor. Em seguida, chame a função exibir_vetor. Por fim, chame a função somar_elementos e armazene o resultado em uma variável para exibi-la.
  4. Compilação: Compile com gcc main.c vetor_utils.c -o programa.
