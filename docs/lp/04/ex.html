Exercício 1: Jogo da Velha Simples

Contexto

Vamos criar a base para um jogo da velha! O objetivo é criar um programa que desenha o tabuleiro na tela e permite que um jogador faça uma jogada. Este exercício é perfeito para entender como os índices de uma matriz correspondem a uma grade visual.

Requisitos

  • Boa prática: Use #define LINHAS 3 e #define COLUNAS 3 para definir o tamanho do tabuleiro.
  • Crie uma matriz de caracteres: char tabuleiro[LINHAS][COLUNAS];.
  • Inicialize a matriz. Use um laço aninhado para preencher todas as posições com um espaço em branco ' '.
  • Crie um trecho de código para exibir o tabuleiro de forma formatada, com as linhas e colunas separadas, como no exemplo.
  • Peça ao usuário para digitar a linha (0 a LINHAS-1) e a coluna (0 a COLUNAS-1) da sua jogada.
  • Marque a posição escolhida na matriz com o caractere 'X'.
  • Exiba o tabuleiro novamente com a jogada realizada.

Passos para a Implementação

  1. Estrutura Inicial: Comece com a estrutura básica de um programa em C, incluindo a biblioteca <stdio.h>, os #define para `LINHAS` e `COLUNAS`, e a função main.
  2. Declaração: Dentro da main, declare a matriz char tabuleiro[LINHAS][COLUNAS]; e duas variáveis int para guardar a linha e a coluna da jogada.
  3. Inicialização do Tabuleiro: Crie um laço for aninhado. O laço externo percorrerá as linhas (de 0 a `LINHAS-1`) e o interno as colunas. Dentro do laço interno, atribua um espaço em branco a cada posição: tabuleiro[i][j] = ' ';.
  4. Exibição do Tabuleiro (1ª vez): Após inicializar, crie outro laço aninhado para mostrar o tabuleiro. Use printf para desenhar as linhas e colunas, exibindo o conteúdo de tabuleiro[i][j] em cada célula.
  5. Entrada do Usuário: Use printf para pedir ao usuário que digite a linha e a coluna. Use scanf para ler os dois números e guardá-los nas variáveis que você criou.
  6. Realizar a Jogada: Acesse a posição da matriz informada pelo usuário e atribua o caractere 'X': tabuleiro[linha][coluna] = 'X';.
  7. Exibição Final: Copie e cole o mesmo código que você usou para exibir o tabuleiro pela primeira vez. Agora ele mostrará o tabuleiro com a jogada 'X' marcada.

Exemplo de Funcionamento

--- Jogo da Velha ---

   0   1   2
0  _ | _ | _ 
1  _ | _ | _ 
2    |   |   

Jogador 'X', sua vez!
Digite a linha (0-2): 1
Digite a coluna (0-2): 1

   0   1   2
0  _ | _ | _ 
1  _ | X | _ 
2    |   |

Exercício 2: Mapa de Assentos do Cinema

Contexto

Você é o responsável pelo sistema de reservas de um pequeno cinema. Sua tarefa é criar um programa que exibe os assentos disponíveis, permite que um cliente escolha um lugar e atualiza o mapa para mostrar que o assento agora está ocupado.

Requisitos

  • Boa prática: Use #define FILEIRAS 5 e #define POLTRONAS 5 para definir o tamanho do cinema.
  • Crie uma matriz de caracteres: char cinema[FILEIRAS][POLTRONAS];.
  • Inicialize todos os assentos como 'L' (Livre).
  • Exiba o mapa de assentos na tela, mostrando as fileiras e poltronas numeradas para facilitar a escolha do usuário.
  • Peça ao usuário para digitar a fileira e a poltrona que deseja reservar.
  • Verifique se o assento escolhido está livre (se o valor na matriz é 'L').
  • Se estiver livre, mude o valor para 'O' (Ocupado) e mostre uma mensagem de sucesso.
  • Exiba o mapa de assentos atualizado.

Passos para a Implementação

  1. Estrutura Inicial: Comece com os #include, os #define para `FILEIRAS` e `POLTRONAS`, e a função main.
  2. Declaração: Declare a matriz char cinema[FILEIRAS][POLTRONAS]; e variáveis para a fileira e poltrona escolhidas pelo usuário.
  3. Inicialização do Cinema: Use um laço aninhado para percorrer a matriz e preencher todas as posições com o caractere 'L'.
  4. Exibição do Mapa: Crie um laço aninhado para exibir o mapa. Use printf para mostrar os números das fileiras e poltronas e o conteúdo de cada posição da matriz.
  5. Entrada do Usuário: Peça para o usuário digitar a fileira e a poltrona desejadas. Lembre-se que o usuário digitará de 1 a 5, mas a matriz usa índices de 0 a 4. Você precisará subtrair 1 dos valores lidos antes de usá-los.
  6. Verificação e Reserva: Use um if para checar o conteúdo da matriz na posição escolhida: if (cinema[fileira-1][poltrona-1] == 'L').
    • Se for 'L', mude para 'O' e imprima uma mensagem de sucesso.
    • Caso contrário (else), imprima uma mensagem dizendo que o assento está ocupado.
  7. Exibição Final: Mostre o mapa atualizado, repetindo o código de exibição.

Exemplo de Funcionamento

--- Mapa de Assentos ---
  P1 P2 P3 P4 P5
F1 L  L  L  L  L 
F2 L  L  L  L  L 
F3 L  L  L  L  L 
F4 L  L  L  L  L 
F5 L  L  L  L  L 

Digite a fileira desejada (1-5): 3
Digite a poltrona desejada (1-5): 4

Assento reservado com sucesso!
