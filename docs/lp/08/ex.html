Exercícios Práticos: Funções em C

Pratique a modularização e a reutilização de código.

1. Mensagem de Boas-Vindas

Crie uma função chamada saudacao que não recebe nenhum parâmetro e não retorna nada (void). A única coisa que essa função deve fazer é imprimir a mensagem "Bem-vindo ao programa!". Na sua função main, chame a função saudacao.

2. Dobro de um Número

Crie uma função chamada calcularDobro que recebe um número inteiro como parâmetro e retorna o dobro desse número. Na função main, peça ao usuário para digitar um número, passe-o para a função e exiba o resultado retornado.

3. Média de Três Números

Crie uma função chamada calcularMedia que recebe três notas do tipo float como parâmetros e retorna a média aritmética delas. Na main, chame esta função com três valores e imprima o resultado.

4. Verificador de Maioridade

Crie uma função chamada ehMaiorDeIdade que recebe uma idade (inteiro) como parâmetro. A função deve retornar 1 se a idade for maior ou igual a 18, e 0 caso contrário. Na main, peça uma idade ao usuário, chame a função e, usando um if, imprima "Acesso permitido" ou "Acesso negado" com base no retorno.

5. Tabuada

Crie uma função imprimirTabuada que recebe um número inteiro como parâmetro. A função deve usar um laço for para imprimir a tabuada de multiplicação daquele número (de 1 a 10). A função não precisa retornar nada (void).

6. Desenhando Linhas

Crie uma função chamada desenharLinha que recebe um número inteiro tamanho como parâmetro. A função deve imprimir na tela uma linha com a quantidade de traços (-) igual ao tamanho informado. Chame essa função na main para desenhar linhas de diferentes tamanhos.

7. Maior de Dois Números

Crie uma função chamada encontrarMaior que recebe dois números inteiros como parâmetro e retorna o maior entre eles. Na main, peça dois números ao usuário, passe para a função e exiba o resultado.

8. Conversor de Temperatura

Crie uma função celsiusParaFahrenheit que recebe uma temperatura em Celsius (float) e retorna o valor correspondente em Fahrenheit. A fórmula é: F = C * 1.8 + 32. Teste a função na main.

9. Exibindo uma Ficha

Crie uma struct Carro com os campos char modelo[50] e int ano. Crie uma função exibirCarro que recebe uma variável do tipo struct Carro como parâmetro (passagem por valor) e exibe seus dados de forma formatada. Na main, crie e preencha os dados de um carro e passe-o para a função.

10. Cadastro de Turma com Função

Crie um programa que cadastre 3 alunos em um vetor de structs. O preenchimento de cada aluno deve ser feito por uma função separada que retorna uma struct preenchida.

Passos Sugeridos:

  1. Crie uma struct Aluno com nome e nota.
  2. Crie uma função chamada criarAluno. Ela não recebe parâmetros, mas retorna uma struct Aluno.
  3. Dentro de criarAluno, declare uma variável local do tipo struct Aluno, peça ao usuário para digitar o nome e a nota, e guarde nessas variáveis. Ao final, retorne a struct preenchida com o comando return.
  4. Na função main, declare um vetor de struct Aluno de tamanho 3 (ex: turma[3]).
  5. Crie um laço for que se repita 3 vezes. A cada volta, chame a função criarAluno e armazene o resultado na posição correta do vetor (ex: turma[i] = criarAluno();).
  6. Após o primeiro laço, crie um segundo laço para percorrer o vetor turma e exibir os dados de todos os alunos cadastrados.
